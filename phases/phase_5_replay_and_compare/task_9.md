# Task 9: Export Comparison as Markdown

## Phase 5 — Replay & Compare

**Status:** Not Started

## Description

Generate a shareable markdown report from a replay comparison, suitable for sharing with a team or filing a bug report.

## Requirements

- `probe compare N M --export comparison.md` writes a markdown file
- Report includes: original request params, both responses (truncated to 500 chars), token/cost/latency table, diff summary
- Include a "generated by probe" footer with version and timestamp
- Markdown renders well on GitHub and Notion
- Also available from dashboard: "Export comparison" button downloads the .md file
